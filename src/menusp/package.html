This package contains contains all the classes that make up the menu system, and is used by all the menu applications. The menu items are contained in 4 tables in the dba2 Oracle schema, namely DMN_MENU_ITEMS, DMN_PARAMETERS, DMN_PARAM_SELECT and DMN_SCRIPT_TYPE. Any application that uses tables out of another Oracle schema, must have synonyms with the same names for these 4 tables.

All applications using this package should have the following:

  • A args2 String array defined, e.g. String[] args2 = { "papplication=EdmMenuP", "powner=edm", "pttitle=EDM", "put=1", // 1 == admin user, 2 == normal user "pdate=27 May 2002" };
      with
    • papplication = the name of the application using the package
    • powner = the name of the Oracle schema
    • pttitle = the title that will be displayed in the top bar
    • put = the user type of the menu system
    • pdate = the date the application was last updated which appears in the bottom bar
  • The Tables.userid and Tables.password variables must be set, e.g.
    menusp.Tables.userid = "edm";
    menusp.Tables.password = "ter91qes";
